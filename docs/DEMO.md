# Demo run-of-show (~3 minutes)

Goal: show real autonomy and the emergent re-planning, fast, with zero risk of a broken demo.

## Before you start

- `npm run build && npm start` (or open the deployed Scalingo URL).
- Agora runs demo-safe with no API key or Redis — nothing here depends on the network.
- Keep the browser at ~1440px wide so the three-column control room is fully visible.

## Script

**1. The hook (20s).**
"Personalized 1:1 tutoring is the most effective thing we know of in education — and the hardest
to scale. We built six AI agents that do the whole tutoring loop autonomously. Watch."

**2. Launch autopilot (15s).**
On the launcher, leave mode on **Autopilot**, leave the learner level around the middle, and
click **Recursion**. Say: "I'm giving it one input — a goal — and nothing else."

**3. Narrate the autonomy (45s).**
Point at the three columns as they animate:
- *Plan board* (top): "The Planner just decomposed the goal into concepts."
- *Agent crew* (left): "Each agent lights up as it acts — Tutor teaching, Assessor checking."
- *Activity stream* (right): "This is every autonomous decision, live."
- *Mastery dial*: "Mastery climbs as concepts are passed."

**4. The money moment — emergent remediation (40s).**
Wait for the red **Diagnostician** event: *"Prerequisite gap detected — re-planning."* Point at
the plan board: a new card appears, tagged **auto-added**.
"Nobody planned that lesson. The learner missed a check, the Diagnostician traced it to a missing
prerequisite, and the Orchestrator rewrote its own plan to fix it — then it goes back and
re-teaches the original concept. That's the agentic behavior we care about."

**5. Completion (15s).**
When it finishes: "Goal reached autonomously, end-to-end — it even tells you it re-planned to
close a gap." Point at the summary: concepts mastered, average mastery, auto-remediations, run
time.

**6. Human-in-the-loop variant (25s).**
Click **New run → Interactive → Adding fractions**. Answer one quiz yourself.
"Same autonomous system; here a human answers the checks while the agents still plan, teach,
diagnose and adapt. Autonomy with a human in the loop when you want one."

**7. Close (20s).**
"It's a production-shaped Node + Redis app: real-time over SSE and Redis pub/sub, deployable to
Scalingo in one command, tested end-to-end. And with an API key, the same agents teach with a
live LLM — but the demo you just saw needs no network at all, by design."

## If asked "is the AI real?"

Be honest and precise: the **orchestration, planning, diagnosis, and adaptation are real
autonomous logic**. The teaching/feedback *text* is generated by a live LLM when an API key is
configured; in the demo it runs from a deterministic pedagogical engine so the run is repeatable
and offline-safe. The agent decisions — what to teach next, when to remediate — are made by the
system either way.
